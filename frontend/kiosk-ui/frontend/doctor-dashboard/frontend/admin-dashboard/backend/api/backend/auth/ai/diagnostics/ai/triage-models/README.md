# Sanjivani – AI-Powered Rural Health Kiosks

Sanjivani is an offline-first, Edge AI–powered healthcare kiosk system designed
to deliver affordable primary healthcare in rural and underserved regions.

## Live Prototype
https://sanjivani-health-kiosks-56d5fc7a.base44.app/

## Key Features
- AI-assisted vitals analysis and diagnostics
- Multilingual, low-literacy patient kiosk interface
- Doctor teleconsultation and triage dashboard
- Admin dashboard for district-level health surveillance
- Offline-first Edge AI inference

## System Architecture
- Frontend: Patient kiosk UI, doctor portal, admin dashboard
- Backend: Secure APIs and authentication
- AI Layer: Edge AI diagnostic and triage models

## Project Status
Prototype-stage implementation. Code modules will be added in the next phase.

## Roadmap
- Phase 1: UI and workflow prototype (completed)
- Phase 2: Device and Edge AI integration
- Phase 3: Pilot deployment
- Phase 4: National scale rollout
